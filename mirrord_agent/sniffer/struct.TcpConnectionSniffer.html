pub async fn new( receiver: Receiver<SnifferCommand>, network_interface: Option<String> ) -> Result<Self, AgentError>

Creates and prepares a new TcpConnectionSniffer that uses BPF filters to capture network packets.

The capture uses a network interface specified by the user, if there is none, then it tries to find a proper one by starting a connection. If this fails, we use “eth0” as a last resort.
